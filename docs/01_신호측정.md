# 1단계: 신호 측정 가이드

목표: 빈 LED(또는 부저) 단자가 결주 때 내보내는 신호의 **전압**과 **극성**을 알아낸다.
이 값으로 ESP32 입력 보호회로(분압저항)를 정한다. **ESP32 GPIO는 3.3V까지만 안전**하다.

## 준비물
- 디지털 테스터(멀티미터)
- 악어클립 또는 가는 점퍼선

## 측정 순서

### A. GND(기준선) 찾기
1. 테스터를 DC 전압(V⎓) 20V 범위로.
2. 검은 프로브(COM)를 장치의 **배터리 마이너스/전원 GND**에 댄다.
3. 이게 모든 측정의 기준이 된다.

### B. 대기 전압 측정 (결주 없을 때)
1. 장치 전원 ON, 정상(결주 없음) 상태로 둔다.
2. 빨간 프로브를 **빈 LED 단자 각 핀**에 하나씩 대본다.
3. 각 핀의 전압을 기록. (보통 2핀: 한쪽은 +전원 상시, 다른쪽이 신호)

### C. 결주 시 전압 변화 측정 ★핵심★
1. 프로브를 B에서 찾은 **신호로 의심되는 핀**에 고정.
2. **일부러 결주를 발생**시킨다 (씨앗 통 비우거나 센서 앞을 손으로 막았다 떼기 등, 평소 알림 뜨던 조작).
3. 결주 순간 전압이 어떻게 변하나 관찰:
   - 0V → 3.3/5/12V 로 **올라감** = HIGH 액티브 (active-high)
   - 3.3/5/12V → 0V 로 **떨어짐** = LOW 액티브 (active-low)

## 기록표 (채워서 알려주세요)

| 항목 | 값 |
|---|---|
| 신호 핀 대기 전압 | ___ V |
| 신호 핀 결주 시 전압 | ___ V |
| 극성 | HIGH 액티브 / LOW 액티브 |
| 신호가 유지되는 시간 | 순간(짧게 삑) / 결주 동안 계속 |
| 나머지 핀 전압 | ___ V (전원선으로 추정) |

## 안전 주의
- 측정 중 두 핀을 프로브로 **동시에 이으면 안 됨**(쇼트).
- 12V가 나오면 ESP32에 직접 물리면 안 됨 → 분압저항 필수(측정 후 값 계산해 드림).
- 부저 단자는 톤(PWM) 신호라 테스터에 불안정한 값이 뜰 수 있음 → 그래서 LED 단자 우선.
